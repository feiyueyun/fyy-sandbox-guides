# FYY + Docker / Podman

Run FYY CLI and AI Agents in standard OCI containers.

> 🚧 Guide content coming with FYY CLI v1.0-alpha (2026 Q3).

## Prerequisites

- Docker 24+ or Podman 4+
- FYY sandbox base image: `feiyueyun/fyy-sandbox:latest`

## Quick Start

```bash
docker pull feiyueyun/fyy-sandbox:latest
docker run -it feiyueyun/fyy-sandbox:latest bash
```
