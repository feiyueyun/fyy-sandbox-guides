# FYY + Kubernetes RuntimeClass

Run FYY CLI and AI Agents with Kubernetes RuntimeClass-based pod sandboxing.

> 🚧 Guide content coming in Phase 2 (2027 Q1).

## Prerequisites

- Kubernetes cluster with RuntimeClass support
- gVisor or Kata Containers configured as alternative runtime
