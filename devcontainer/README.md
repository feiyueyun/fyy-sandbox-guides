# FYY + Devcontainer

Run FYY CLI and AI Agents in Devcontainer environments (VS Code / GitHub Codespaces).

> 🚧 Guide content coming with FYY CLI v1.0-alpha (2026 Q3).

## Prerequisites

- VS Code with Dev Containers extension, or GitHub Codespaces
- FYY sandbox base image
