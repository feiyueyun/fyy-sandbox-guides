# FYY + gVisor (runsc)

Run FYY CLI and AI Agents with gVisor user-space kernel isolation.

> 🚧 Guide content coming with FYY CLI v1.0-alpha (2026 Q3).

## Prerequisites

- gVisor / runsc installed
- Docker or containerd configured with runsc runtime
